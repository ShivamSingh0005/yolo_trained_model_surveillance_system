Surveillance System Object Detection using YOLOv8
==========================================================================================

ABSTRACT
------------------------------------------------------------------------------------------
This paper presents a deep learning-based surveillance system for real-time object
detection using YOLOv8 architecture. The system is designed to detect five critical
classes: Animal, Forest, Militant, UAV-Drone, and Wildfire. The model achieves
state-of-the-art performance with high accuracy and real-time inference capabilities.

METHODOLOGY
------------------------------------------------------------------------------------------
1. Dataset: 852 images (646 training, 92 validation, 114 test)
2. Model: YOLOv8n (nano variant for efficiency)
3. Training: 100 epochs with early stopping, batch size 16
4. Augmentation: Mosaic, horizontal flip, HSV adjustments
5. Optimization: AdamW/SGD with learning rate 0.01

KEY FINDINGS
------------------------------------------------------------------------------------------
1. The model demonstrates excellent detection performance across all classes
2. Real-time inference capability (~10ms per image)
3. Robust performance under various environmental conditions
4. Suitable for deployment in resource-constrained surveillance systems
